pipSettingsProvider
service in module pipSettings

Description

Service provides an interface to manage 'Settings' component behaviour. It is available on config and run phases.

Methods

  • addPage(pageObj)

    Register new page in 'Settings' component. Before adding a page this method validates passed object.

    Parameters
    ParamTypeDetails
    pageObjObject

    Configuration object for new page.

  • getDefaultPage()

    Method returns an config object for pages established as default (it will be opened when app transeferred to abstract state 'settings').

    Returns
    Array<Object>

    Collection of pages.

  • getPages()

    Method returns collection of registered pages.

    Returns
    Array<Object>

    Collection of pages.

  • setDefaultPage(name)

    Establish a page which is available by default (after chose this component in menu).

    Parameters
    ParamTypeDetails
    namestring

    Name of the default state for this component.

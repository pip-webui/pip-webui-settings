pipSettingsPageController
controller in module pipSettings

Description

The controller is used for the whole settings tabs and provides navigation menu on the left and load content into right panel. This component is integrated with 'pipAppBar' component and adapt the tabs header. The component has predefined states 'settings.base_info' and 'settings.active_sessions'. Each of these states require user's authorization.

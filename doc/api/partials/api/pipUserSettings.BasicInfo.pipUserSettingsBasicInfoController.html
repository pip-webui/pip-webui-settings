pipUserSettingsBasicInfoController
controller in module pipUserSettings

Description

Controller for the predefined 'basic_info' state. Provides sync changes user's profile with remote profile. On state exit everything is saved on the server.

Methods

  • onChangeBasicInfo()

    Saves changes onto server. This method responses on change of the input information. It is updated user's party profile. Also it updates user's profile in $rootScope.
